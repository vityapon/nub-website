«Esthetic Furniture»
  Сайт представляет из себя интернет-магазин мебели с небольшим ассортиментом товаров, которые можно приобрести по низким ценам.
  Цель работы: закрепить навыки написания кода сайта.
  Для разработки сайта была выбрана программа «Visual Studio Code», так как он бесплатен для пользования, хорошо оптимизирован и имеет большое количество плагинов.

Дизайн-макет и визуальное оформление.
  При создании макетов использовался сайт «Figma», им можно пользоваться бесплатно, и у него удобный и понятный интерфейс.
  Интерфейс ресурса выполнен в соответствии с утвержденным фирменным стилем:
    - Цветовая палитра: преобладание теплой цветовой гаммы.
    - Типографика: в качестве основного шрифта для набора текста утвержден гарнитурный шрифт «Franklin Gothic Medium».
