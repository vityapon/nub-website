# Esthetic Furniture
Сайт представляет из себя интернет-магазин мебели с небольшим ассортиментом товаров, которые можно приобрести по низким ценам.

Цель работы: закрепить навыки написания кода сайта.

Для проекта я составил техническое задание и реализовывал сайт следуя ему.

# Дизайн-макет и визуальное оформление
Для разработки сайта была выбрана программа «Visual Studio Code», так как он бесплатен для пользования, хорошо оптимизирован и имеет большое количество плагинов.

При создании макетов использовался сайт «Figma», им можно пользоваться бесплатно, и у него удобный и понятный интерфейс.

Интерфейс ресурса выполнен в соответствии с утвержденным фирменным стилем:
- Цветовая палитра: преобладание теплой цветовой гаммы.
- Типографика: в качестве основного шрифта для набора текста утвержден гарнитурный шрифт «Franklin Gothic Medium».

# Составляющие сайта
Шапка
- Официальное наименование сайта (логотип).
- Тематическое графическое изображение (герб, фото продукта или иной утвержденный бренд-ассет).
- Корпоративный слоган или миссия организации.
- Блок ключевой информации.

Каталог товаров
- Наименование товара.
- Цена за единицу.
- Развернутое описание характеристик и назначения.
- Интерактивный элемент управления («Добавить в корзину») для инициации процесса формирования заказа.

Заключительный блок
- «О компании»: ссылки на страницы с подробными сведениями об организации.
- «Часто задаваемые вопросы (FAQ)»: раздел для оперативного получения ответов на типовые вопросы пользователей.
- «Обратная связь»: официальный канал связи со службой поддержки, представленный адресом электронной почты для запросов.
